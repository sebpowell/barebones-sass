Components

Barebones includes a series of basic components which are easily configurable. Each component includes on the bare minimum in terms of styling, so you don't need to spend time undoing properties.

Styling Components

Components are styled using a user sass map. For example, $modal-user. To style a component, simply create a new map and specify your settings.
