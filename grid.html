Grid

Container

To create a grid, use the .row class (clears the floats), then .columns-* classes for different column widths. Columns will grow to 100% of the container width at the breakpoint specified in your grid map.

<div class="columns">
 <div class="columns-4"></div>
 <div class="columns-4"></div>
 <div class="columns-4"></div>
</div>

Mobile columns

Helper classes are also available to create different grids for different screen sizes.

<div class="columns">
 <div class="columns-4-device-x-small columns-5-device-medium"></div>
 <div class="columns-4-device-x-small columns-3-device-medium"></div>
 <div class="columns-4-device-x-small columns-4-device-medium"></div>
</div>

Offsets

Offsets allow you to shift a column to the right by a specified number of columns.

<div class="columns">
	<div class="columns-3 offset-9"></div>
</div>
