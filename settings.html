Introduction

Barebones is a boilerplate to help you build your own UI framework. It is not a fully-fledged UI kit or library (Bootstrap, Foundation etc), but a set of tools common to all projects; a grid system and commonly used utility classes and mixins reponsive helpers, , put together in a convenient package to kickstart your next project. It is design-agnostic – so you can spend less time overwriting and more time creating.

To provide an example, this is the login modal on Airbnb, recreated using the various classes this is the sort of thing you can very quickly create with Barebones.

Prerequisites

Barebones requires SASS v3.3 or later to compile .scss files into .css. Instructions can be found here.

Additionally, vendor prefixing mixins are not used or included as you can use the autoprefixer plugin to do this.

Get started

1. Install the package

Barebones is available as a Bower package. If you want to skip package control and copy the SASS files directly, you can do that too – simply download the latest release from GitHub and integrate the source files directly. 

bower install barebones-sass --save

2. Create a settings file

In your project folder, create a _settings.scss file (or whatever you want to call it) with the following values. These are various settings Barebones uses – a more detailed explanation of each option can be found here.

$bb-border-radius-scale: 5;

$bb-breakpoints: ();

$bb-colors: ();

$bb-letter-spacing: ();

$bb-line-heights: ();

$bb-global-delimiter: "-";

$bb-helpers: ();

$bb-hover-class-name: "hover";

$bb-max-width-scale: 14;

$bb-responsive: true;

$bb-shadows: ();

$bb-spacing-scale: 10;

$bb-spacing-base: 4px;

$bb-spacing-sides: ();

$bb-text-sizes: ();

3. Create your main SCSS file

he final step is to import all these files into your main files that will be compiled into actual CSS. When you're done, it should look something like this:

// Always import the settings file first
@import _settings.scss;

// Always import the settings file first
@import bower-components/barebones-sass/dist/_base.scss;

Settings

Barebones is fully and easily customisable. Note that if you use the 'packaged' version (see installation guide), variables are not prefixed with 'bb-' (for example, $bb-border-radius-scale would simply be $border-radius-scale).

Border-radius scale

Sets the scale for .border-radius-* classes. 

$bb-border-radius-scale: 5;

// This will generate border-radius helpers for values 1 - 5.
.border-radius-1 {
	border-radius: 1px;
}

...

Breakpoints

Stores your breakpoints. This is used by the media-query mixin, and by the helper generator to generate responsive classes.

Access using the breakpoints($value) function.

$bb-breakpoints: (
	"mobile": 480px,
	"tablet": 720px,
	"desktop": 1200px
);

Colors

Stores your colour palette in a organised way. Used by the background-color, text-color and fill helper generators. Access using colors($category, $value).

$colors: (
	"brand": "#157AFB",
	"secondary": (
		"orange": "#F8B000",
		...
	),
	"greyscale": (
		"50": "#FAFAFA",
		"100": "#F5F5F5",
		...
	),
	"keyline": "#E5E5E5",
);

// The above values can be accessed like this.
.element {
	background-color: colors("brand", "primary");
}

Letter-spacing

$bb-spacing-base: (
	"small": "0.01em",
	"medium": "0.5em",
	"large": "0.1em"
);

Line-heights

Used by the line-height helper generator. 

$bb-line-heights: (
	"small": 1,
	"medium": 1.5,
	"large": 2
);

Global delimiter

Specifies how classes are named. 

$bb-global-delimiter: "-";

// Output
.background-cover {
	background-size: cover;
}


$bb-global-delimiter: "_";

// Output
.background_cover {
	background-size: cover;
}

Grid

Settings for the grid system.

columns Number of columns
gutter Width between columns
breakpoint Breakpoint at which point column widths change to 100% of viewport
mobile-spacing Default marin-bottom value for collapsed columns. 
$grid: (
	columns: 12,
	gutter: "24px", 
	breakpoint: "medium",
	mobile-spacing: "24px"
);

Hover class name

Specify the name for hover classes. Used by background-color, text-color and fill generators.

$bb-helper-class-name: "hover";

// Output
.background-white-hover:hover {
	background-color: white;
}

Responsive classes

Barebones will automatically generate responsive classes using it's helper generator and specified breakpoints. You can disable this by setting the value to false.

$bb-breakpoints: (
	"medium": 780px;
);

// Viewports larger than 'medium'.
@media max-width(780px) {
	.display-block-above-medium {
		display: block;
	}
}

// Viewports smaller than 'medium'.
@media min-width(780px) {
	.display-block-above-medium {
		display: block;
	}
}

Grid

Container

To create a grid, use the .row class (clears the floats), then .columns-* classes for different column widths. Columns will grow to 100% of the container width at the breakpoint specified in your grid map.

<div class="columns">
 <div class="columns-4"></div>
 <div class="columns-4"></div>
 <div class="columns-4"></div>
</div>

Mobile columns

Helper classes are also available to create different grids for different screen sizes.

<div class="columns">
 <div class="columns-4-device-x-small columns-5-device-medium"></div>
 <div class="columns-4-device-x-small columns-3-device-medium"></div>
 <div class="columns-4-device-x-small columns-4-device-medium"></div>
</div>

Offsets

Offsets allow you to shift a column to the right by a specified number of columns.

<div class="columns">
	<div class="columns-3 offset-9"></div>
</div>

Mixins

arrow($color, $size, $direction)

Creates a triangle, used for things like tooltips or dropdowns.

Option Values / Example
$direction
  • "up"
  • "down"
  • "left"
  • "right"
  • "top-right"
  • "top-left"
  • "bottom-right"
  • "bottom-left"
$color Any hex value (e.g. #E5E5E5)
$size Any px value (e.g. 6px) 
.element { 
	&:after {
		@include arrow("down", "black", "20px");
	}
 }

aspect-ratio($width, $height, $container: optional)

Creates a triangle, used for things like tooltips or dropdowns.

.element { 
	@include aspect-ratio(16, 9);
}

// Output
.aspect-ratio-16-9 { 
	position: relative; 
}

.aspect-ratio-16-9:before { 
	display: block; 
	content: ""; 
	width: 100%; 
	padding-top: 56.25%; 
}

.aspect-ratio-16-9 > .aspect-ratio-content { 
	position: absolute; 
	top: 0; 
	left: 0; 
	right: 0; 
	bottom: 0; 
}

background-image($width, $height, $container: optional)

Creates a triangle, used for things like tooltips or dropdowns.

.element { 
	@include aspect-ratio(16, 9);
}

clearfix()

Clears floats. Apply this to the parent element.

.element {
	@include clearfix;
}

// Output
.clearfix { zoom: 1; }

.clearfix:before, .clearfix:after { content: ""; display: table; }

.clearfix:after { clear: both; }

ellipsis($width)

Use this to truncate text once it reaches a certain width.

.element {
	@include ellipsis(300px);
}

helpers($name, $css...)

@include helpers("background-center", "background-position: center");

// Output
.background-cover { background-size: cover; }

@media (min-width: 780px) { .background-cover-above-medium { background-size: cover; } }

@media (max-width: 780px) { .background-cover-below-medium { background-size: cover; } }

no-bullet()

Removes list-style-type, margin and padding attributes from ul elements (useful when creating navigation components).

ul {
	@include no-bullet;
}

map-deep-get($values...)

Fetch any value from a nested map. You can go as deep as you need to. 

$colors: (
	ui: (
		error: (
			light: #e6e6e6;
		)
	)
);


ul {
	color: map-deep-get($colors, ui, error, light);
}

Media-query

You can reference values in your breakpoints map. It will also accept values that aren't defined.

.element {
	@include if-above(device-medium) {
		color: red;
	}
}

.element {
	@include if-above(800px) {
		color: blue;
	}
}

.element {
	@include if-below(device-small) {
		color: blue;
	}
}

.element {
	@include if-between(800px, 1200px) {
		color: blue;
	}
}

Placeholder

Changes <input> placeholder text color.

.element {
	@include placeholder {
		color: colors("greyscale", 500);
	}
	&:focus {
		@include placeholder {
			color: colors("greyscale", 300);
		}
	}
}

circle($size);

Create a square or circle. 

.element {
	@include circle(200px);
}

// Output
.element {
	width: 200px;
	height: 200px;
	border-radius: 200px;
}

square($size);

Create a square.

.element {
	@include square(400px);
}

// Output
.element {
	width: 400px;
	height: 400px;
}

text-uppercase($letter-spacing)

Transforms text to uppercase, accepts an argument to specify letter-spacing.

.element {
	@include text-uppercase(0.1em);
}

strip-units($value)

Transforms text to uppercase, accepts an argument to specify letter-spacing.

.element {
	@include uppercase(0.1em);
}
