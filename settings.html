Settings

Border-radius scale

Sets the scale for .border-radius-* classes. 

// This will generate border-radius helpers for values 1 - 5.
$bb-border-radius-scale: 5;

.border-radius-1 {
	border-radius: 1px;
}

Breakpoints

Stores your breakpoints. This is used by the media-query mixin, and by the helper generator. You can name these however you like, and have as many or as few as you want.

Access using the breakpoints($value) function.

$bb-breakpoints: (
	"small": 480px,
	"medium": 780px,
	"large": 1200px
);

Colors

Stores your colour palette in a organised way. Used by the background-color, text-color and fill helper generators. Access using colors($category, $value).

$colors: (
	"brand": #157AFB,
	
	"secondary": (
		"orange": #F8B000,
		...
	),
	
	"greyscale": (
		"50": #FAFAFA,
		"100": #F5F5F5,
		...
	),

	"ui": (
		"error": (
			"light": #E5E5E5,
			"dark": #E5E5E5
		),
		...
	),
	
	"keyline": #E5E5E5,
);

// The above values can be accessed like this.
.element {
	background-color: colors("keyline");
}

.element {
	background-color: colors("brand", "primary");
}

.element {
	background-color: colors("ui", "error", "light");
}

Letter-spacing

Used by the letter-spacing helper generator.

$bb-letter-spacing: (
	"small": "0.01em",
	"medium": "0.5em",
	"large": "0.1em"
);

Line-heights

Used by the line-height helper generator. 

$bb-line-heights: (
	"small": 1,
	"medium": 1.5,
	"large": 2
);

Grid

Settings for the grid system.

container

Name for the columns container.
columns

Number of columns
gutter

Width between columns
breakpoint

Breakpoint at which point column widths change to 100% of viewport, as specified in your $breakpoints map.
mobile-spacing

Default marin-bottom value for collapsed columns.

 
$grid: (
	"container": ".columns",
	"columns": 12,
	"breakpoint": "tablet",
	"gutter": 24px,
	"mobile-spacing": 24px
);

Hover class name

Specify the name for hover classes. Used by background-color, text-color and fill generators.

$bb-helper-class-name: "hover";

// Output
.background-white-hover:hover {
	background-color: white;
}

Max-width scale

Specifies number of variants for the max-width helper.

$bb-max-width-scale: 14;

// Output
.max-width-100 {
	max-width: 100px;
}

...

.max-width-1400 {
	max-width: 1400px;
}

Responsive classes

Barebones will automatically generate responsive classes using it's helper generator and specified breakpoints. You can disable this by setting the value to false.

$bb-breakpoints: (
	"medium": 780px;
);

// Viewports larger than 'medium'.
@media max-width(780px) {
	.display-block-above-medium {
		display: block;
	}
}

// Viewports smaller than 'medium'.
@media min-width(780px) {
	.display-block-above-medium {
		display: block;
	}
}

Shadows

Used to keep spacing consistent throughout the project. The function will multiply uour Access using shadows($shadow).

$bb-shadows: (
	"small": rgba(),
	"medium": rgba(),
	"large": rgba()
);

Spacing base

Used to keep spacing consistent throughout the project. This the base unit used by the margin, padding, top, left, right and bottom helpers to generate their variants.

will multiply uour Access using sizes($size). Used 

$bb-spacing-base: 5px;

.element {
	padding: sizes(2);
}

// Will output
.element {
	padding: 10px;
}

Spacing scale

Used to keep spacing consistent throughout the project. The function will multiply uour Access using sizes($size). Used by margin, padding, top, left, right and bottom helpers.


$bb-spacing-base: 5px;

.element {
	padding: sizes(1);
}

Text-sizes

Used by the text-size generator to generate classes for each specified text size.

$bb-text-sizes: (
	"14": "11px",
	"18": "18px",
	"24": "24px",
	"32": "32px",
	"48": "48px"
);
