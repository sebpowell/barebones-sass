Breakpoints

Used to generate responsive helpers and for the responsive mixins.

Values

Stored in $breakpoints map. Add additional values to this map.

small 480px
medium 780px

Functions

Create rules depending on the You can reference values in your breakpoints map, will also accept values that aren't defined in your $breakpoints map.

.element {
	@include if-above(medium) {
		color: red;
	}
	
	@include if-below(800px) {
		color: blue;
	
	@include if-between(800px, 1200px) {
		color: blue;
	}
}
