Mixins

Arrow

Creates a triangle, used for things like tooltips or dropdowns. Available directions are "up", "right", "down" and "left".

.dropdown-menu {
 &:after {
  @include arrow($direction, $color, $size);
 }
}

Clearfix

Clears floats. Apply this to the parent element.

.row {
 @include clearfix;
}

Ellipsis

Adds ellipsis to text prevent text overflowing.

.label {
 @include ellipsis($max-width);
}

Media Queries

By default, media queries are mobile-first (the styles specified in the mixin will only apply to device sizes greater than or equal to the specified breakpoint). The available values are those specified in the $breakpoints map. Another mixin is available if you want to target screen sizes below a certain size.

p {
  font-size: 16px;
  @include respond-above(device-large) {
    font-size: 20px;
  }
}

No Bullet

Removes list-style-type, margin and padding attributes from ul elements.

ul {
  @include no-bullet;
}

Uppercase

Sets text-transform to uppercase and adds 0.1em letter-spacing. No arguments required.

h3 {
 @include uppercase;
}
