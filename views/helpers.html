Helpers

Background Color

Generates a background-color helper for all colors specified in the colors map.

<div class="background-primary">...</div>
<div class="background-secondary">...</div>
<div class="background-greyscale-light">...</div>
<div class="background-greyscale-dark">...</div>

Borders

Adds a keyline to any element. The following variants are available:

  • border-left
  • border-top
  • border-right
  • border-bottom
  • border-all
  • no-border
<div class="border-top">...</div>
 <div class="border-left">...</div>
 <div class="no-border">...</div>

Margin & Padding

Margin helpers are particularly useful to help fine tune spacing in between elements, without needing to write custom selectors. The helper classes are available for both padding and margin.

  • padding-{direction}, margin-{direction} followed by the unit value

    • quarter
    • half
    • base
    • double-half
    • double
    • triple
    • .no-margin / .no-padding removes any padding or margins.
