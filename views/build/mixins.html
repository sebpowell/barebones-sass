Mixins

Arrow

Creates a triangle, used for things like tooltips or dropdowns. Available directions are "up", "right", "down" and "left". Accepts the following arguments arrow($direction, $color, $size).

.element {
  &:after {
   @include arrow(down, black, 20px);
  }
 }

Containers

Containers are used to center and restrict content width on the page. They are fluid, and the width is limited using the max-width property. They also include left and right padding to give breathing room on smaller devices. By default this is using the mobile-padding value from the settings file.

You can create a container using the container($max-width) function.

.element {
  @include container($max-width);
 }

Clearfix

Clears floats. Apply this to the parent element.

.element {
  @include clearfix;
 }

Ellipsis

Adds ellipsis to limit text width.

.element {
  @include text-overflow($width);
}

Media Queries

Use respond-above($device) and respond-below($device) to target different viewport sizes defined in your settings file.

.element {
  @include respond-above(device-medium) {
    color: red;
  }
}

No Bullet

Removes list-style-type, margin and padding attributes from ul elements (useful when creating navigation components).

ul {
 @include no-bullet;
}
