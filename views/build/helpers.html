Helpers

Helpers or utlity classes should generally be used sparingly. They server two main purposes: first, for rapid brower prototyping, and second for adpating an element or component within a given context.

Background Color

Generates a background-color helper for all colors specified in the palette category in the theme map.

.background-brand-primary {
 background-color: #F5F5F5 !important;
}

Borders

Adds or removes a keyline to a given element. The border color uses the keyline value fromm the theme map.

The following helpers are avaiable:

  • border-left
  • border-top
  • border-right
  • border-bottom
  • border-all
  • no-border
.border-top {
 background-color: #F5F5F5 !important;
}

Margin & Padding

Margin helpers are particularly useful to help fine tune spacing in between elements, without needing to write custom selectors. The helper classes are available for both padding and margin.

  • padding-{direction}, margin-{direction} followed by the unit value

    • quarter
    • half
    • base
    • double-half
    • double
    • triple
    • .no-margin / .no-padding removes any padding or margins.
