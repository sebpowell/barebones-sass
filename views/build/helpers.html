Helpers

Helpers or utlity classes should generally be used sparingly. All helper class attributes are suffixed with an !important flag to avoid specificity issues.

Background Color

Generates a background-color helper for all colors specified in the palette category in the settings map. To append the colour category and name to the helper. 

<div class="background-brand-primary"></div>
.background-brand-primary {
 background-color: #F5F5F5 !important;
}

Borders

Adds or removes a keyline to a given element. The border color used is the keyline value specified in the settings map.

  • border-left
  • border-top
  • border-right
  • border-bottom
  • border-all
  • no-border
<div class="background-brand-primary"></div>
.background-brand-primary {
 background-color: #F5F5F5 !important;
}
