Containers are used to center and restrict content width on the page. Containers also include left and right padding to give breathing room on smaller devices.

Containers are fluid, and the width is limited using the max-width property. Various sizes are generated from a range of values specified in components/container.scss:

Colors

These are the colors used to generate button colours and various helpers. These are categorised as follows. You can of course add additional categories.

To access these colours, use the colours() function.

.element {
  background: colours(brand, primary);
 };

Containers2

Every layout grid starts with a container that wraps around the grid. Creating a new container is very straightforward.

This sets a max-width, as defined in your $theme file and centers the container across the screen. It also applies a clearfix to clear fixes from any possible floated columns you place within it. 

.wrapper {
  @include container;
}
// Compiled CSS
.wrapper {
 max-width: 1000px;
}

Creating Your Own

Barebones also allows you to create your own layout containers.

By default, columns collapse at 720px. 

.example-sidebar {
 @include span(6, last);
 @include respond-above(device-small) {
   @include span(8, last);
 }
}
