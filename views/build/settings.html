Settings

Barebones keeps all global settings in a single SASS (_settings.sass). This map is a starting point â€“ you can extend it any way you like.

The theme($args...) function allows you to navigate the map and retrieve any value you want. It accepts any number of arguments, allowing you to navigate nested maps with ease.

.element {
  border-right: theme(colours, keyline, light);
  color: theme(colours, brand, primary);
  z-index: theme(layers, element);
}

A helper function is also provided for each category for faster access.

Colours

Access values using colours($category, $value).

  colours: (
    brand: (
      primary: #157AFB,
      secondary: #FE5607,
      ...
    ),
    secondary: (
      orange: #F8B000,
      ...
    ),
    greyscale: (
      50: #FAFAFA,
      100: #F5F5F5,
      ...
    ),
    keyline: (
      light: #E5E5E5,
      ...
    ),
  )

Fonts

Specify font familys and respective weights. You can access using the fonts($family) and weights($weight) functions.

  fonts: (
    primary: (
      family: Open Sans,
      weights: (
        light: 100,
        regular: 200,
        semi-bold: 500
      )
    secondary: (
      family: Futura PT,
      weights: (
        bold: 600,
      )
   ),

Grid

Specify global grid settings. Access using themes(layout, grid, $setting) or grid($setting).

   grid: (
    columns: 12,
    site-width: 1200px,
    gutter: 1/4,
    gutter-type: margin,
    mobile-padding: 24px,
   ),

Breakpoints

Specify global grid settings. Access using themes(layout, grid, $setting) or grid($setting).

   breakpoints: (
    device-small: 320px,
    device-medium: 480px,
    ...
   ),

Sizes

Used to keep spacing consistent throughout the project. Access using sizes($size).

   sizes: (
    quarter: 4px,
    half: 8px,
    base: 16px,
    ...
   ),

Layers

Stores z-indexes for different components so you can easily see stacking order. Access using layers($element).

   layers: (
    modal: 5,
    backdrop: 2,
    navigation: 4,
    ...
   ),
