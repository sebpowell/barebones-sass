The Theme File

The theme file is a global configuration file where you can specify various project settings. The following settings are included by default. This map is broken down into various parts.

Colours

Used to generate various helpers and components (e.g. buttons).

  colours: (
    brand: (
      primary: red,
      secondary: green,
   ),
   secondary: (
     purple: purple,
     cyan: cyan,
   ),
   greyscale: (
    brand: 50,
    brand2: 100,
   ),
   keyline: (
     light: light,
     dark: dark,
   ),

Fonts

Specify font familys and respective weights.

  fonts: (
    primary: (
      family: Times New Roman,
      weights: (
        light: 100,
        regular: 200,
      )
    primary: (
      family: Times New Roman,
      weights: (
        light: 100,
        regular: 200,
      )
   ),

Grid

Specify font familys and respective weights.

   layout: (
    brand: (
      primary: red,
      secondary: green,
   ),
   secondary: (
     purple: purple,
     cyan: cyan,
   ),
Category Required
Colours Yes – colours are used to generate buttons and various helpers.

Accessing Values

The themes() function allows you to navigate the map and retrieve any value you want. This map is a starting point – you can extend it any way you like.

.element {
  border-right: themes(colours, keyline, light);
  color: themes(colours, brand, primary);
  z-index: themes(layers, element);
}
