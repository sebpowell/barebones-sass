Colours

Colors and their shades and tints are stored in config/colors.scss and are accessed using the colors($category, $value) function. The first argument specifies the category, the second (optional) returns the specified shade or tint.


// Returns a lighter version of the primary brand color.
.object {
  background-color: colors(brand-primary, light);
}

// Returns the base primary brand color.
.object {
  background-color: colors(brand-primary);
}

Layout Units

Layout units are used to keep spacing consistent throughout the project. Default values are stored in config/sizing.scss and are multiples of the base font size (16px). These values can be accessed using the sizes() function.

Key Value
quarter 4px
half 8px
base 16px
double-half 24px
double 32px
triple 48px
quadruple 64px

Stacking

All z-index values are stored in config/z-index. The value is determined by the element's position in the list. Items lower on the list have higher z-indexes. To create a new element, add it to the list, then access the value using the layer() function.

Note to self: Use map instead of mixin! 

.element {
 z-index: layer(modal);
}
