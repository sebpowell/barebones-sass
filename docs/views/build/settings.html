Settings

Barebones stores all project settings in a structured SASS map (_settings.scss). This map is a starting point – you can extend it any way you like. Default categories are included below (or click here to see an example).

A helper function is provided to access each map. Please note that some sections are required by mixins, so you muist have them – your SASS won't compile otherwise. Others are optional and depend on your specific needs.

Import Settings

Allows you to specify class prefixing (by default Barebones classes aren't namespaced), and to exclude certain file categories (for example, if you only want a grid).

Colours

Access values using either settings(colours, $category, $value) or colours($category, $value).

  colours: (
    brand: (
      primary: #157AFB,
      secondary: #FE5607,
      ...
    ),
    secondary: (
      orange: #F8B000,
      ...
    ),
    greyscale: (
      50: #FAFAFA,
      100: #F5F5F5,
      ...
    ),
    keyline: (
      light: #E5E5E5,
      ...
    ),
  )

Fonts

Specify font familys' and their respective weights. Access using fonts($family) and weights($family, $weight) functions.

  fonts: (
    primary: (
      family: Open Sans,
      weights: (
        light: 100,
        regular: 200,
        semi-bold: 500
      )
    ),
    secondary: (
      family: Futura PT,
      weights: (
        bold: 600,
      )
    ),

Grid

Specify global grid settings. Access using settings(layout, grid, $setting) or grid($setting).

   grid: (
    columns: 12,
    site-width: 1200px,
    gutter: 1/4,
    gutter-type: margin,
   ),

Breakpoints

Specify global breakpoints. It's generally best to set these as device sizes, rather than target specific devices. Access using settings(layout, breakpoints, $size) or breakpoints($size).

   breakpoints: (
    device-small: 320px,
    device-medium: 480px,
    ...
   ),

Sizes

Used to keep spacing consistent throughout the project. Access using settings(layout, sizes, $size) or sizes($size).

   sizes: (
    quarter: 4px,
    half: 8px,
    base: 16px,
    ...
   ),

Layers

Stores z-indexes for different components so you can easily see the stacking order. Access using settings(layout, layers, $component) or layers($component).

   layers: (
    modal: 5,
    backdrop: 2,
    navigation: 4,
    ...
   ),
