Settings

Barebones stores all project settings in a structured SASS map (_settings.scss). This map is a starting point – you can extend it any way you like.

To access values, use the theme($args...) function. It accepts any number of arguments, allowing you to navigate nested maps with ease.

.element {
  border-right: theme(colours, keyline, light);
  color: theme(colours, brand, primary);
  z-index: theme(layers, element);
}

A helper function is also provided for each category for faster access.

Colours

Access values using either themes(colours, $category, $value) or colours($category, $value).

  colours: (
    brand: (
      primary: #157AFB,
      secondary: #FE5607,
      ...
    ),
    secondary: (
      orange: #F8B000,
      ...
    ),
    greyscale: (
      50: #FAFAFA,
      100: #F5F5F5,
      ...
    ),
    keyline: (
      light: #E5E5E5,
      ...
    ),
  )

Fonts

Specify font familys' and their respective weights. Access using fonts($family) and weights($family, $weight) functions.

  fonts: (
    primary: (
      family: Open Sans,
      weights: (
        light: 100,
        regular: 200,
        semi-bold: 500
      )
    ),
    secondary: (
      family: Futura PT,
      weights: (
        bold: 600,
      )
    ),

Grid

Specify global grid settings. Access using themes(layout, grid, $setting) or grid($setting).

   grid: (
    columns: 12,
    site-width: 1200px,
    gutter: 1/4,
    gutter-type: margin,
    mobile-padding: 24px,
   ),

Breakpoints

Specify global breakpoints. It's generally best to set these as device sizes, rather than target specific devices. Access using themes(layout, breakpoints, $size) or breakpoints($size).

   breakpoints: (
    device-small: 320px,
    device-medium: 480px,
    ...
   ),

Sizes

Used to keep spacing consistent throughout the project. Access using themes(layout, sizes, $size) or sizes($size).

   sizes: (
    quarter: 4px,
    half: 8px,
    base: 16px,
    ...
   ),

Layers

Stores z-indexes for different components so you can easily see the stacking order. Access using themes(layout, layers, $component) or layers($component).

   layers: (
    modal: 5,
    backdrop: 2,
    navigation: 4,
    ...
   ),
