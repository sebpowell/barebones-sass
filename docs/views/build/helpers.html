Helpers

Helpers or utlity classes should generally be used sparingly. All helper class attributes are suffixed with an !important flag to avoid specificity issues.

Background Color

Generates a background-color helper for all colors specified in the palette category in the settings map. To append the colour category and name to the helper.

<div class="background-brand-primary"></div>
.background-brand-primary {
 background-color: #F5F5F5 !important;
}

Borders

Adds or removes a keyline to a given element. The border color used is the keyline value specified in the settings map.

border-left
border-top
border-right
border-bottom
border-all
no-border 
<div class="border-bottom"></div>
.border-bottom {
 border-bottom: 1px solid #F5F5F5 !important;
}

Margin & Padding

Generated from the $sizes map.

padding-{direction}-{size} Applies padding in the specified direction of the specified size.
margin-{direction}-{size} Applies padding in the specified direction of the specified size.
no-margin Removes all margins
no-padding Removes all padding 
<div class="margin-bottom-double"></div>
.background-brand-primary {
 margin-bottom: 32px !important;
}

Grid

Rows & Columns

To create a grid, use the .row class (clears the floats), then .column-{number} classes for different column widths.

Columns 4
Columns 5
Columns 3
<div class="columns">
 <div class="columns-4">
 <div class="columns-5">
 <div class="columns-3">
</div>

Helper classes are also available to create different grids for different breakpoints.

Columns 4
Columns 5
Columns 3
<div class="columns">
 <div class="columns-4-device-small">
 <div class="columns-5-device-small">
 <div class="columns-3-device-small">
</div>

Offsets

Offsets allow you to shift a column to the right by a specified number of columns.

Columns 4
<div class="columns">
 <div class="columns-4 offset-8">
</div>

Galleries

Galleries are grids containing a repeating number of equal-sized items. They work in a similar way to columns.

They're slightly different to columns in that you can create odd numbers of items – for example, in a 12-column grid, you mayb want a gallery with 5 items on each row.

<div class="columns">
 <div class="columns-4">
 <div class="columns-5">
 <div class="columns-3">
</div>

Visibility

We provide easy to use classes to hide content on specific screen sizes. Helpers are generated from the $breakpoints map.

<div class="is-hidden"></div>
<div class="is-hidden-above-device-small"></div>
<div class="is-hidden-below-device-small"></div>
.is-hidden {
 display: none !important;
}
