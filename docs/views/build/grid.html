Grid

Barebones includes a flexible, responsive grid system. It's a semantic grid, allowing you to target given components – for example, a sidebar – without littering your HTML with helper classes and unnecessary divs.

Grid Settings

Specify your grid settings in your settings file. The following settings are available.

columns (default: 12) 12 Number of columns in your grid
breakpoint (default: device-medium – 720px) The point at which all column sizes should expand to 100%.
gutter (default: 1/4) Expressed as a calculated as a fraction of the column widthns as such: 1/8, 1/4 etc.
grid-type (default: margin) Specify whether you want to create gutters using padding or margin.

Columns

First, you'll need to create a container.

Containers

Containers are used to center and restrict content width on the page. They are fluid, and the width is limited using the max-width property. They also include left and right padding to give breathing room on smaller devices, using the mobile-padding value from the settings file.

You can create a container using the container($max-width) function.

$max-width container max-width
$mobile-gutter (true false) removes the mobile gutter on smaller screens (for example, if you're nesting a container inside another container). 
.element {
  @include container(600, $mobile-gutter: false);
 }


.container {
  @include container(1000px);
}

Use the span($columns, $position, $gutter) function to specify column widths. To create different widths at different breakpoints, use the respond-above($device) mixin.

$columns Number of columns the component spans, of the total number defined in your settings file.
$position (optional) if the column is the last one, pass the 'last' keyword to remove the margin.
$gutter (optional) You can remove the gutter by passing the keyword 'none'. 
.sidebar {
  @include span(4); // Creates a column that spans 4 colums
}

.content {
  @include span(8, $position: last); // Creates a column that spans 8 columns
}

Galleries

Galleries are made up of elements that repeat across multiple lines. Here's how you would create one.

.photo-gallery {
  @include clearfix;
  // Clears the floats.
}

.photo-gallery-item {
	@include gallery(3);
	// Creates a gallery with each item taking up 3 columns of your grid (on a 12 grid-layout, this would be 4 items).
}
