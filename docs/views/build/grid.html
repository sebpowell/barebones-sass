Grid

Barebones includes a flexible, responsive grid system that can be used in one of two ways using a range of settings specified in your settings file. Gutters are created by applying margin-right to every column, except for the final one (where the margin is removed using the last-child rule).

columns (default: 12) Number of columns
breakpoint The point at which all column sizes should expand to 100%.
gutter Expressed as a calculated as a fraction of the column widthns as such: 1/8, 1/4 etc.
offset

Grid Generator

The span($columns, $position) grid generator allows you to target a given component – for example, a sidebar – without littering your markup the need of 'messy' helper classes.

$columns Number of columns the component spans, of the total number defined in your settings file.
$position (optional) if the column is the last one, pass the 'last' keyword to remove the margin.
$gutter (optional) You can remove the gutter by passing the keyword 'none'. 
.sidebar {
  @include span(4);
}

.content {
  @include span(8, $position: last);
}

To create different widths at different breakpoints, use the respond-above($device) mixin.

.example-sidebar {
 @include span(8);
 @include respond-above(device-small) {
   @include span(6);
 }
}

Rows & Columns

To create a grid, use the .row class (clears the floats), then .column-{number} classes for different column widths.
