Grid

Barebones includes a flexible, responsive grid system. It's a semantic grid, allowing you to directly target clasesA lightweight semantic grid framework for Sass and Bourbon.allows you to target a given component – for example, a sidebar – without littering your markup the need of 'messy' helper classes. it relies entirely on Sass mixins and does not pollute your HTML with presentation classes and extra wrapping div's.

.sidebar {
	@include span(4); // Creates a container div that spans 4 colums
}

.content {
   @include span(8, $position: last); // Creates a container div that spans 4 colums
	 @include respond-above(medium) {
	   @include span(6);
	 }
	}

Specify your grid settings in your settings file. The following settings are available.

columns (default: 12) Number of columns in your grid
breakpoint The point at which all column sizes should expand to 100%.
gutter Expressed as a calculated as a fraction of the column widthns as such: 1/8, 1/4 etc.
grid-type (margin (default) | padding) Specify whether you want to create gutters using padding or margin.

Greating a layout

Create a container

.container {
  @include container(1000px);
}

Create a Layout

The span($columns, $position) grid generator

To create different widths at different breakpoints, use the respond-above($device) mixin.

$columns Number of columns the component spans, of the total number defined in your settings file.
$position (optional) if the column is the last one, pass the 'last' keyword to remove the margin.
$gutter (optional) You can remove the gutter by passing the keyword 'none'. 
| .example-sidebar {
|  @include span(8);
|  @include respond-above(device-small) {
|    @include span(6);
|  }
| }
| .sidebar {
|   @include span(4);
| }
| 
| .content {
|   @include span(8, $position: last);
| }

Galleries

Galleries are made up of elements that repeat across multiple lines. Here's how you would create one. 

.photo-gallery {
	@incluce clearfix;
	// Clears the floats. 
}

.photo-gallery-item {
	@include gallery(3);
	// Creates a gallery with each item taking up 3 columns of your grid (on a 12 grid-layout, this would be 4 items).
}
