Grid

Barebones includes a flexible, responsive grid system that can be used in one of two ways.

  • Grid generator
  • Helper classes

Grid Generator

The grid generator allows you to target the specific component (e.g. a sidebar) without the need of 'messy' helper classes.

The Barebones grid works in the following way:

  • Each column has right margin applied to it. Margins are % based and are calculated as a fraction of the column width. So you'll need to express margins as such: 1/8, 1/4 etc. This margin is removed on the last column
  • Columns automatically grow to 100% of the container width at the breakpoint defined in the SASS map (theme, breakpoints, grid).

using the span($columns, $position) function. They work in the fol

  • $colums: number of columns spanned by the element.
  • $position: if the column is the last one, pass the 'last' keyword to remove the margin.
.sidebar {
  @include span(4);
}

.content {
  @include span(8, last);
}

To create different widths at different breakpoints, you can couple this with the respond-above($device) mixin.

.example-sidebar {
 @include span(6, last);
 @include respond-above(device-small) {
   @include span(8, last);
 }
}

Helper Classes

Rows & Columns

To create a grid, use the .row class (clears the floats), then .column-{number} classes for different column widths.

Columns work in the following way:

Columns 5
Columns 3
<div class="columns">
 <div class="columns-4">
 <div class="columns-5">
 <div class="columns-3">
</div>

Helper classes are also available to create different grids for different breakpoints.

Columns 5
Columns 3
<div class="columns">
 <div class="columns-4 columns-4-device-small">
 <div class="columns-5 columns-5-device-small">
 <div class="columns-3 columns-3-device-small">
</div>

Galleries

Galleries are grids containing a repeating number of equal-sized items. They work in a similar way to columns
