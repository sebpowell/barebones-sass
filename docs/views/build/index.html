Getting Started

Barebones is a boilerplate to help you quickly build your own front-end framework. It is not a fully-fledged library, but a set of commonly used tools; a grid system, helper classes and a range of mixins, put together in a convenient package to kickstart your next project.

The best way to use Barebones is to download the source from Github and start building!

Download

Bower

You can also get the latest release using Bower, although this is somewhat against the philsophy behind this project (rather than include as a depenency, you should ideally use this as a foundational code-base).

If you decide to do this, you'll need to do the following:

1. Install the Barebones package

bower install barebones

2. Create a settings file (see here for more info)

3. Create a new SASS file, import the settings file first followed by _base.sass.

@import "_settings.scss"
@import "bower_components/dist/_base"

Requirements

Barebones relies heavily on SASS maps to work, so you'll need to make sure you're using SASS 3.3 or higher. Additionally, vendor prefixing mixins are not used or included as you can use the autoprefixer plugin to do this.
