Mixins

Arrow

Creates a triangle, used for things like tooltips or dropdowns.

$direction "up", "right", "down" or "left"
$color for example, #E5E5E5
$size for example, 6px 
.element {
  &:after {
   @include arrow(down, black, 20px);
  }
 }

Placeholder

Changes form placeholder text color.

.element {
  @include placeholder(grey);
   &:focus {
     placeholder(lightGrey);
   }
 }

Uppercase

Containers

Containers are used to center and restrict content width on the page. They are fluid, and the width is limited using the max-width property. They also include left and right padding to give breathing room on smaller devices, using the mobile-padding value from the settings file.

You can create a container using the container($max-width) function.

$max-width container max-width
$mobile-gutter (true | false) removes the mobile gutter on smaller screens (for example, if you're nesting a container inside another container). 
.element {
  @include container(600, $mobile-gutter: false);
 }

Clearfix

Clears floats. Apply this to the parent element.

.element {
  @include clearfix;
 }

Ellipsis

Adds ellipsis to limit text width.

.element {
  @include text-overflow($width);
}

Media Queries

Use respond-above($device) and respond-below($device) to target different viewport sizes defined in your settings file.

.element {
  @include respond-above(device-medium) {
    color: red;
  }
}

No Bullet

Removes list-style-type, margin and padding attributes from ul elements (useful when creating navigation components).

ul {
 @include no-bullet;
}
