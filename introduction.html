Introduction

Barebones is fully customisable SCSS boilerplate to help you build your own UI framework.

It is not a UI kit or library – Bootstrap, Foundation and the like – but a set of commonly used tools common put together in a convenient package to kickstart your next project:

  • Grid
  • Mixins
  • Responsive 'composer' classes

Why do I need this?

Barebones allows you to quickly build a custom, fully responsive UI. Here's an exmple of how you could use Barebones to replicate the Airbnb login modal.

Prerequisites

Barebones requires SASS v3.3 or later to compile .scss files into .css. Vendor prefixing mixins are not used as you can use autoprefixer to do this.

Get started

1. Install the package

Barebones is available as either a Bower or NPM package.

bower install barebones-sass --save
npm install barebones-sass --save-dev

If you want to skip package control, you can do that too – download the latest release from GitHub and integrate the source files directly.

2. Create a settings file

In your project folder, create a _settings.scss file (or whatever you want to call it) with the following values. Each setting is covered in the following section.

$bb-border-radius-scale: 5;

$bb-breakpoints: ();

$bb-colors: ();

$bb-letter-spacing: ();

$bb-line-heights: ();

$bb-global-delimiter: "-";

$bb-helpers: ();

$bb-hover-class-name: "hover";

$bb-max-width-scale: 14;

$bb-responsive: true;

$bb-shadows: ();

$bb-spacing-scale: 10;

$bb-spacing-base: 4px;

$bb-spacing-sides: ();

$bb-text-sizes: ();

Barebones also allows you to specify your own naming convention. For each generated class, you can decide how that class will be outputted. 

// These are the settings used by the align-items helper generator to 
// create classes for the flex align-items property. 
$bb-align-items: (
	"modifiers": (
		"flex-start": "items-flex-start",
		"flex-end": "items-flex-end",
		"center": "items-center"
	)
);

// This will output
.align-items-flex-start { 
	flex-items: flex-start; 
}

You may want a less verbose version, and you ay want to specify your own class names. A short name boilerplate with all the relevant variables nams is aavailable here.

// You can make this less verbose by changing the names: 
$bb-align-items: (
	"modifiers": (
		"flex-start": "items-fs",
		"flex-end": "items-fe",
		"center": "items-c"
	)
);

// Will output: 
.items-fs {
	flex-items: flex-start;
}

3. Create your main SCSS file

Import these files into your main SCSS file so they can compiled.

Note that the import order matters!

// Always import the settings file first
@import _settings.scss;

// Optional – import the _module-names.scss if you're using it
@import _module_names.scss;

// Always import the settings file first
@import bower-components/barebones-sass/dist/_base.scss;
